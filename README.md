# MERN E-Commerce Website

This is a full-stack e-commerce web application developed using the MERN stack.  
The project was designed, customized, and implemented as part of my MERN stack learning journey and internship submission.

The application focuses on building a complete frontend experience along with backend API integration, following real-world e-commerce architecture.

## Developed By
**Snehal Labade**

## Tech Stack
- React.js (Frontend UI & Routing)
- Redux (State Management)
- Node.js (Backend Runtime)
- Express.js (REST API Development)
- MongoDB (Database)
- Cloudinary (Image Management)

## Key Features
- Responsive and user-friendly UI
- Product listing and product detail pages
- Advanced filters (price, category, ratings)
- User authentication (login & registration UI)
- Shopping cart functionality
- Order flow structure
- Admin panel structure for product and order management
- Secure API architecture using JWT

## Project Highlights
- Modular and scalable folder structure
- Clean separation of frontend and backend
- Redux for global state handling
- API-based data flow between frontend and backend
- Custom branding and UI modifications

## Note
This project is developed for learning and internship evaluation purposes.  
Backend services and database connectivity are environment-dependent during local execution.

The project demonstrates practical understanding of MERN stack concepts, application structure, routing, and frontend-backend integration.
